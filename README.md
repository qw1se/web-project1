# Quote Randomizer

A simple web application that displays random inspirational quotes. Built with plain HTML, CSS, and JavaScript.

---

## Demo
View the live demo:  
https://qw1se.github.io/web-project1

---

## Overview
Quote Randomizer is a lightweight app with a minimal interface. Click a single button to fetch and display a new random quote. The layout is responsive and works well on different screen sizes.

---

## Features
- Random quote generation
- Clean, responsive design
- One-click interface to load a new quote

---

## Installation & Local Run
1. Clone the repository:
   git clone https://github.com/qw1se/web-project1.git
2. Open `index.html` in your browser (double-click or serve via a local web server).

Note: This is a static project — no backend or build step is required.

---

## Project Structure
- index.html — main HTML markup  
- styles.css — styling and layout  
- script.js — quote fetching and display logic

---

## Technologies
- HTML5  
- CSS3  
- JavaScript (ES6+)
